Project Discription:

This is the project Piano Notation Helper. The inspiration for this project was that most members of our team have a musical background. So, we wanted to integrate our musical and technical background to do something good for the people by building a musical application. The function of the project is to teach people, whoever wants to learn music theory, and staff notations corresponding to a piano note.

Throughout the process of coding, we utilized multiple programming languages. Those include but not limited to Java, Javascript, Android Studio Development, HTML, CSS, P5.js. We utilized these languages, integrated each of our team members' code and built the final application.

There are several challenges during the process. Considering that we were not very proficient in android development kit, most of us did not have a clear vision on the development process. However, we still managed to utilize the knowledge of our programming languages, communicated ideas with each other, learned things on the go and built the application. Within a short span of time, our team is very proud to learn the plethora of things by working together.

Future developments:

Right now, at its demo stage, the application only allows you to practice and learn one song. In future development we would like to add more songs with varying difficulties. We would also like to add a stronger user interface as well as allowing the user to eventually create their own staff notation.
